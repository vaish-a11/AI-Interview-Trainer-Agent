# AI-Interview-Trainer-Agent
Project Title :education and career development 
Problem Statement : interview trainer agent
Objective :
Develop an AI-powered Interview Trainer Agent that:
Generates technical interview questions 
Generates HR interview questions 
Evaluates answers 
Provides feedback 
Identifies skill gaps 
Improves interview readiness

Technologies Used:
Lang Flow 
IBM watsonx.ai 
IBM Granite Models 
Python 
IBM cloud
I used this approach 
                   Lang Flow + IBM Granite + IBM watsonx.ai


Workflow :
 the complete flow is given in the ppt
Features:
1. Chat Input
Receives user queries and interview requests.
2. Agent
Acts as the central AI component that processes user inputs and generates responses.
3. IBM Granite Model
Provides natural language understanding, reasoning, and interview question generation.
4. Chat Output
Displays generated interview questions, feedback, and recommendations.
5. Prompt Template
Structures prompts to guide the AI in producing accurate and consistent responses.
6. Read File (Future Enhancement)
Enables resume file reading and content extraction for personalized interview preparation

Future Scope:
Resume PDF Analysis
Generate personalized questions from resumes.
RAG-Based Knowledge Base
Retrieve interview-related content.
Voice-Based Mock Interviews
Real-time interview simulation.
Performance Analytics
Track improvement over time.
Multi-Language Support
Support interviews in multiple languages.
